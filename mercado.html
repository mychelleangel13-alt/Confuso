⚙️ Como isso funciona por trás dos panos

A leitura do save (decriptação Easy Save 3 + parse do inventário) roda inteiramente no seu navegador, em JavaScript — por isso nada é enviado pra fora. Já a consulta de preço (steamcommunity.com/market/priceoverview) não libera CORS pra chamadas diretas do navegador, então essa página chama um pequeno proxy de servidor que só repassa a pergunta pra Steam e devolve a resposta com o header liberado. Veja o arquivo proxy/steam-price-proxy.js entregue junto com o site — é o código pronto desse proxy, pra você publicar de graça num Cloudflare Worker.

Configure a URL do seu proxy publicado na constante PROXY_URL no topo do script desta página.
